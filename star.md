##### 1. Используя только символы `()[]!+` получите строку с буквой `u`. Количество символов `()[]!+` неограничено.

```js
console.log(...)
// 'u'
```

<details><summary><b>Решение</b></summary>
<p>

```js
console.log((!![] + [])[+!![] + +!![]])
```
